# `@zivex/laravel-vite-translations`

Core npm package for the Laravel Vite Translations project.

## What this package contains

- Vite plugin export from `@zivex/laravel-vite-translations`
- Runtime API from `@zivex/laravel-vite-translations/runtime`
- Framework adapters for React, Vue, and Svelte
- ESLint plugin export
- SSR preload helpers
- CLI entrypoint: `laravel-vite-translations`

## Main entrypoints

- `@zivex/laravel-vite-translations`
- `@zivex/laravel-vite-translations/runtime`
- `@zivex/laravel-vite-translations/react`
- `@zivex/laravel-vite-translations/vue`
- `@zivex/laravel-vite-translations/svelte`
- `@zivex/laravel-vite-translations/eslint`
- `@zivex/laravel-vite-translations/preload`

## Build

```bash
pnpm build
```

## Publish

This package is intended to be published from the `packages/core` directory.

```bash
npm publish
```
